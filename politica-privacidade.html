Política de Privacidade e Cookies

Última atualização: Abril 2026

Este site é gerido por Joana Clarinha (a seguir "eu" ou "responsável pelo tratamento"), prestadora de serviços de Thai Yoga Massage em Lisboa. Contacto: joanaclarinha@icloud.com

1. Que dados recolho

O único dado pessoal que este site recolhe é através do formulário de contacto, onde podes partilhar:

  • Nome
  • Email ou número de telemóvel
  • Mensagem

Não recolho dados de pagamento, dados de saúde, nem qualquer outra informação sensível através do site.

2. Para que uso os dados

Os dados partilhados no formulário são usados exclusivamente para responder à tua mensagem e, se assim o desejares, para marcar uma sessão. Não são usados para fins comerciais nem partilhados com terceiros, exceto o processador técnico do formulário (ver ponto 4).

3. Base legal

O tratamento dos dados do formulário de contacto baseia-se no teu consentimento, expresso ao submeter o formulário (Artigo 6.º, n.º 1, al. a) do RGPD).

4. Terceiros envolvidos

Formspree — o formulário de contacto é processado pela plataforma Formspree (formspree.io), que recebe temporariamente os dados para os encaminhar para o meu email. A Formspree age como subcontratante e está sujeita à sua própria política de privacidade.

5. Cookies e rastreamento

Este site pode utilizar cookies de análise (Google Analytics) para compreender como os visitantes navegam no site. Estes cookies só são ativados com o teu consentimento explícito, através do banner que aparece na primeira visita.

Se aceitares os cookies de análise, o Google Analytics recolhe dados anónimos como: páginas visitadas, tempo de visita, localização aproximada e tipo de dispositivo. Estes dados não te identificam pessoalmente.

Podes retirar o teu consentimento a qualquer momento, limpando os cookies do browser ou recusando no banner.

6. Quanto tempo guardo os dados

Os dados do formulário de contacto são guardados apenas durante o tempo necessário para responder ao teu pedido, tipicamente até 12 meses. Podes pedir a eliminação a qualquer momento.

7. Os teus direitos (RGPD)

Tens direito a:

  • Aceder aos teus dados pessoais
  • Corrigir dados incorretos
  • Pedir a eliminação dos teus dados
  • Opor-te ao tratamento
  • Apresentar reclamação à CNPD (Comissão Nacional de Proteção de Dados)

Para exerceres qualquer um destes direitos, contacta-me em joanaclarinha@icloud.com.

8. Alterações a esta política

Esta política pode ser atualizada ocasionalmente. A data de última atualização está indicada no topo desta página.
